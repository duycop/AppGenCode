Tool đẻ ra code cho Lập trình web

Author: Đỗ Duy Cốp

Create: 2024-10-17

Ý tưởng:

  1. Tạo table trên Sql Server bằng SQL Managerment Studio
  2. Script table as \ CREATE to \ New windows => Copy SQL => Paste tab input SQL
  3. Bấm nút GenCode
  4. Sẽ tự động tạo SP với các tham số phù hợp cho các @action, đầu ra trả về json
  5. Sẽ tự động tạo mã HTML: Có dùng jQuery, jConfirm, Bootstrap
  6. Sẽ tự động tạo mã CSS: Tạo một số mã css đơn giản
  7. Sẽ tự động tạo mã JS có dùng lệnh của jQuery, jConfirm: tạo form nhập, gửi dữ liệu qua $.ajax tới api, nhận về json, tiền xử lý ra html và hiển thị
  8. Sẽ tự động tạo mã api.aspx.cs để nhận dữ liệu gửi lên, gọi db để ra json, gửi lại client
